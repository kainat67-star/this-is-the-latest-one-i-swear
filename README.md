# Neo — Multi-channel ad analytics

Vite + React + TypeScript dashboard for advertising performance across Google Ads, Meta, TikTok, Snapchat, and X.

## Scripts

- `npm install` — install dependencies  
- `npm run dev` — start dev server  
- `npm run build` — production build  
- `npm run preview` — preview production build  

## Stack

React Router, Tailwind CSS, shadcn/ui, Recharts, Framer Motion, TanStack Query.
